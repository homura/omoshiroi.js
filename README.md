

## Usage

```js
import { SekiroDangerousPallet } from './sekiro';

const dangerous = new SekiroDangerousPallet();
const view = dangerous.getView();
document.body.appendChild(view);
dangerous.play();
```

## Example

```
git clone https://github.com/homura/omoshiroi.js.git
npm run example
```
